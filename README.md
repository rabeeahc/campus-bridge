# Campus Bridge

Campus Bridge is a platform designed to help international students transition smoothly into university life.

It brings together everything a student needs before arriving, during their studies, and while living in a Kocaeli all in one place.

## Vision

To make every international student feel prepared, connected, and supported from the moment they receive their university acceptance letter.

## Status

Currently in development.

## Tech Stack

- React
- TypeScript
- Node.js
- Express
- PostgreSQL
- Prisma
- Tailwind CSS

## MVP

- Arrival Guide
- University Hub
- City Guide
- Community
- Interactive Map
